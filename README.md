Project Overview

This project focuses on analyzing hospital patient data to uncover key healthcare insights related to patient admissions, discharge trends, length of stay, treatment costs, and hospital performance metrics. The objective is to help healthcare administrators make data-driven decisions that improve patient care and operational efficiency.

Business Problem
Hospitals generate large volumes of patient data every day. Without proper analysis, it becomes difficult to identify trends, optimize resources, and improve patient outcomes.

This project aims to:

Monitor patient admission and discharge trends
Analyze treatment costs and hospital revenue
Track patient demographics and disease patterns
Measure hospital performance metrics
Support strategic healthcare decision-making
